# How to Start the Frontend Server

## Step 1: Open Terminal/PowerShell

Navigate to this folder (`frontend`) in your terminal.

## Step 2: Start the Server

Run this command:

```bash
npm run dev
```

## Step 3: Wait for Output

You should see something like:

```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

## Step 4: Open Browser

Open http://localhost:5173 in your browser.

---

**Note:** Make sure you're in the `frontend` folder when running `npm run dev`!

If you get errors, make sure:
1. You're in the `frontend` directory
2. Dependencies are installed: `npm install`
3. No other process is using port 5173
